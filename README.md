# Code2Career_AI - EdTech Platform

## 🎓 Overview

Code2Career_AI is a modern, interactive learning platform dedicated to code career development and professional growth in technology.

A vibrant, future-forward platform designed for students and professionals looking to master coding skills and accelerate their tech careers.

## ✨ Features

- **Interactive Courses** - Comprehensive learning paths for different skill levels
- **Career Guidance** - Resources and mentorship for tech career planning
- **Community Support** - Connect with fellow learners and professionals
- **Modern Design** - Vibrant, responsive UI with deep blue and teal branding
- **Progress Tracking** - Monitor your learning journey and achievements
- **Resource Library** - Access curated articles, videos, and tools
- **Industry Insights** - Stay updated with latest tech trends

## 🚀 Getting Started

### Local Development
```bash
git clone https://github.com/RzLetsCode/code2career-ai.git
cd code2career-ai

# Start a local server
python -m http.server 8001

# Open http://localhost:8001 in your browser
```

### Live Platform
https://rzletscode.github.io/code2career-ai/
